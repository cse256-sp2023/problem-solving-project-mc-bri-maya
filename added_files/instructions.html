Instructional Guide:

What are the options for each permission?

Allow: gives users access to a given permission
Deny: denies access to a given permission and overrides “Allow” when both are checked

What are the different types of Permissions?

The permissions for each user are organized into 6 main categories as listed below. Changes applied to the general category will affect those listed under it, so be careful!

Read:
  • read attributes
  • read extended attributes
  • read permissions
Write:
  • create files/write data
  • create folders/append data
Read_Execute:
  • traverse folder/execute file
  • list folder/read contents
Modify:
  • write attributes
  • write extended attributes
  • delete subfolders and files
  • delete
Specical Permissions
  • change permissions
  • take ownership
Full_Control: *Access to all permissions
